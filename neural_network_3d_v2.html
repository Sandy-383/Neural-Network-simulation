🧠 What is a Neural Network?

A neural network is a program loosely inspired by the human brain. It's made of layers of tiny units called neurons. Each neuron takes in numbers, does a small calculation, and passes a result forward.

Think of it like a factory assembly line — each station does one small job, and together they produce a finished product (a digit prediction).

⚪ What is a Neuron?

Each glowing sphere in the 3D view is a neuron. It receives signals from the previous layer, multiplies each by a weight, adds a bias, then squashes the result through an activation function.

output = ReLU( Σ(weight × input) + bias )
Think of a neuron like a voting judge — it weighs all evidence (inputs), has a personal bias, then decides how strongly to "vote" (activate).

🔗 What are Weights?

Each connection line has a weight — a number that says how important that connection is. Positive weights amplify a signal. Negative weights suppress it. Training = finding the right weights.

Connection strength = weight × activation
Like a dial on a mixing board — turning it up amplifies that channel, turning it down mutes it.

📉 What is Loss?

Loss (also called error) measures how wrong the network's prediction is. If the network says digit "3" has 2% probability but the true answer is "3", that's a big mistake → high loss. As training progresses, loss drops.

Loss = -log( probability of correct class )
Like a golf score — lower is better. The training goal is to minimize it.

⚡ What is an Activation Function?

After summing inputs, we pass the result through an activation function to add non-linearity — without it, stacking layers would be no better than one layer.

ReLU(x) = max(0, x) ← hidden layers
Softmax(x) = eˣ / Σeˣ ← output layer
ReLU is like a light switch — negative? Off. Positive? Let it through. Softmax converts raw scores into probabilities that sum to 100%.

Step 1 — Forward Pass

A digit image (28×28 pixels = 784 numbers) enters the Input layer. Each neuron in every layer computes a weighted sum of its inputs, applies ReLU, and passes results to the next layer. The final Output layer runs Softmax to produce probabilities for digits 0–9.

INPUT → HIDDEN 1 → HIDDEN 2 → OUTPUT (probabilities)

Step 2 — Compute Loss

Compare the network's output (e.g. "I think it's a 7 with 15% confidence") against the true label (it's actually a 3). Cross-entropy loss punishes confident wrong answers harshly. A perfect prediction = loss near 0.

Step 3 — Backpropagation

The error travels backwards through the network. Using calculus (chain rule), we compute: "how much did each weight contribute to the error?" These are called gradients. The glow you see on neurons reflects gradient magnitude.

Step 4 — Weight Update (Gradient Descent)

Every weight is nudged slightly in the direction that reduces loss. The size of the nudge = gradient × learning rate. Do this thousands of times and the network converges to weights that classify digits correctly.

new_weight = old_weight − lr × gradient

🔁 Epoch vs Batch

A batch is a small group of training images processed together (e.g. 64 images). An epoch is one full pass through all training data. Training for 20 epochs means the network sees every image 20 times, refining weights each time.
